# 단방향
```java
Member
@ManyToOne
@JoinColumn(name = "TEAM_ID")
private Team team;
```

# 양방향
```java
Team
@OneToMany(mappedBy = "team")
private List<Member> members = new ArrayList<>();
//new ArrayList<>();로 초기화를 해줘야함. 그래야 nullpointexception이 나질 않음.
```

# 1대다 단방향
```java
@Entity
class Parent {
    @Id @GeneratedValue
    private Long id;

    @OneToMany
    @JoinColumn(name = "parent_id") // Child 테이블의 FK
    private List<Child> children = new ArrayList<>();
}

@Entity
class Child {
    @Id @GeneratedValue
    private Long id;
}
```

# 1대다 양방
```java
@Entity
class Parent {
    @Id @GeneratedValue
    private Long id;

    @OneToMany(mappedBy = "parent")
    private List<Child> children = new ArrayList<>();
}

@Entity
class Child {
    @Id @GeneratedValue
    private Long id;

    @ManyToOne
    @JoinColumn(name = "parent_id")
    private Parent parent;
}
```
